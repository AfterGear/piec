1. Rodzaje Okienek

    • Okienko alert - wyświetlająca okienko dialogowe z informacją i przyciskiem OK
    • Okienko prompt - generująca okno, w którym użytkownik może wpisać wartość, zwracaną następnie do programu
    • Okienko confirm - służy do tego , aby użytkownik coś potwierdził (stąd nazwa confirm)

  2. Konwersja liczb
    Wczytanie zmiennej z użyciem prompt wykonywane jest jako ciąg znaków dlatego konieczna jest konwersji z ciągu znaków na wartości liczbowe aby wykonywać obliczenia

    parseFloat(zmienna) - konwersja zmiennej do typu float
    parseInt(zmienna) - konwersja do typu int (liczbowe)
    toString(zmienna) - konwersja zmiennej do typu łańcuchowego

  3. Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:

    • Nazwa zmiennej musi się zaczynać od litery, znaku "$" lub "_", może zawierać tylko litery (bez polskich znaków), cryfry oraz "$" i "_"
    • Wielkość liter ma znaczenie.
      Rekord i rekord to dwie różne zmienne! Jest to bardzo częste przyczyna błędów w kodzie początkujących programistów
    • Należy nie używać słów kluczowych JavaScript jako nazw zmiennych

  4. Typy zmiennych w JavaScript
    W JavaScript nie ma możliwości definiowania różnych typów zmiennych tak jak w C++, typy zmiennych są definiowane w trakcie ich przypisywania.
    Podstawowe zmienne liczbowe w JavScript to:
    Number -> Typ liczbowy

    • zmienna całkowita
    • zmiennoprzecinkowa


    String -> Typ tekstowy
    Boolean -> Typ logiczny
    Null -> Pusta zawartość
    Undefined -> Typ niezidentyfikowany
    Object -> Obiekt
